# 📊 50 Days SQL Challenge – Day 22

## 🧠 Overview
In Day 22, the focus is on using **window functions for department-level analysis** while retaining row-level details.

Unlike GROUP BY, window functions allow us to compute department-level metrics without collapsing individual employee data.

---

## 📌 Objectives

- Show each employee with average salary of their department  
- Show total salary of each department for every employee  
- Show average performance rating of each department
