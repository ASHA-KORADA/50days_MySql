# 📊 SQL Day 9 – Joins & Data Analysis

## 🚀 Project Overview

This project demonstrates how SQL Joins can be used to analyze real-world datasets, identify missing relationships, and extract meaningful insights from incomplete data.

---

## 🎯 Objectives

* Understand relationships between tables
* Work with incomplete datasets
* Use joins to uncover hidden data gaps

---

## 🧠 Concepts Covered

* INNER JOIN
* LEFT JOIN
* NULL Handling
* Data Relationship Analysis

---

## 📌 Tasks Performed

### ✅ Show employees with valid departments

Used INNER JOIN to filter only employees linked to departments.

### ✅ Show all employees (including missing departments)

Used LEFT JOIN to include all records, even when department data is missing.

### ✅ Find employees without department

Identified records where department is NULL.

### ✅ Find who earns how much

Joined employee and salary tables to map salary details.

### ✅ List employees without salary

Detected employees whose salary data is missing.

---

## 💡 Key Learning

Real-world datasets are often:

* Incomplete
* Inconsistent
* Missing relationships

SQL helps to:
✔ Identify missing data
✔ Analyze relationships
✔ Build a complete data picture
